初始dockerfile

dockerfile 就是用来构建docker镜像的构建文件!命令脚本! 先体验一下!

# 通过这脚本可以生成一个镜像,镜像是一层一层的,脚本是一个个命令,每个命令都是一层

# 创建一个dockerfile文件,名字可以随机 建议dockerfile
# 文件中的内容指令(大写) 参数
FROM centos

VOLUME ["volume01","volume02"]

CMD echo "------end------"

CMD /bin/bash/

# 这里的每一个命令都是镜像的一层
